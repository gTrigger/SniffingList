# SniffingList

### This is a small pet project to be able to track all my perfumes and to quickly check what they smell like.

### Served on Vercel: https://sniffing-list.vercel.app/

### Features:
- **AI-Powered Perfume Search & Card Creation**: Automatically generates and creates a perfume card by its title using OpenRouter's free LLMs (might be pretty slow though).
- **Local Storage Collection**: Stores your entire perfume collection in the browser's `localStorage`.
- **URL State Management**: Saves current filter, search, and language settings directly in the URL.
- **Multi-language Support**: Fully localized interface for different languages via i18n. Supported locales are EN, DE and RU.
- **Import & Export**: Allows uploading and downloading your collection data in JSON format.
- **Test Collection**: A quick way to load a pre-made test collection (my very own collection btw 😊) to instantly check out all features.
- **Accessibility**: Oh yeah, it's *accessible*! Easy to navigate via keyboard, VoiceOver assistant and supported reduced motion preference.
- **Cat Loaders**: Very cute and delightful, wow! 🐈
